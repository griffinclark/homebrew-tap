# Griffinclark Tap

## How do I install these formulae?

`brew install griffinclark/tap/<formula>`

Or `brew tap griffinclark/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "griffinclark/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
